# Case Study — Cloud Modernization & Migration

**Role:** Technical Architect (Lead)
**Timeline:** 2021 – 2024

## Overview
Led the modernization and migration of a portfolio of banking applications to cloud-native AWS/GCP microservices to meet scale, compliance and cost targets.

## Responsibilities
- Conducted application discovery, created migration roadmaps and prioritized services for phased migration.
- Re-architected monolithic Java applications into Spring Boot microservices and containerized them with Docker and Kubernetes.
- Automated infrastructure with Terraform and standardized CI/CD with Jenkins and GitHub Actions.
- Implemented IAM guardrails, Zero Trust design, and compliance automation for SOC2 and PCI DSS.

## Outcome
- Migrated 15+ applications, improving scalability by ~45% and lowering hosting costs by ~30%.
- Reduced release errors by 35% and audit preparation time from weeks to days using reusable templates.
