# Case Study — Risk Engine & Predictive Fraud (CITI Bank)

**Role:** Senior Software Engineer / Architect
**Timeline:** Mar 2021 – Jul 2022

## Overview
Designed and implemented a PostgreSQL/Redis based risk engine combined with Python ML models to provide predictive fraud scoring for transaction streams.

## Responsibilities
- Developed REST APIs and batch jobs with Spring Boot for data ingestion and scoring.
- Integrated Python ML models into Java workflows and scheduled batch retraining.
- Tuned PostgreSQL and Redis performance for sub-second responses under heavy load.

## Outcome
- Increased predictive accuracy by ~35% and reduced processing failures by 30%.
