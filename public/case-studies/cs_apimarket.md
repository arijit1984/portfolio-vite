# Case Study — API Plugin Marketplace (DBS)

**Role:** Advisory System Analyst / Architect
**Client:** Development Bank of Singapore
**Timeline:** Jul 2022 – Sep 2022

## Overview
Architected microservices for an API Plugin Marketplace to accelerate partner onboarding and enhance API visibility.

## Responsibilities
- Designed microservices enabling plugin management, partner onboarding and API provisioning.
- Implemented Python ETL + Spark workflows for data processing and anomaly detection.
- Built dashboards with Grafana and ELK for observability and partner analytics.

## Outcome
- Reduced onboarding time from 4 weeks to under 3 weeks for 50+ partners and improved API visibility across services.
